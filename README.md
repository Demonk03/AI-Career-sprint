# AI Career Coach

Веб-приложение для карьерной диагностики IT-специалистов. Пользователь проходит 3 сессии — AI синтезирует ответы и возвращает персональный анализ и план развития.

## Что делает

- **Сессия 1 — Диагностика:** цели, ресурсы, личная эффективность, SWOT
- **Сессия 2 — Функционал:** таблица задач с оценкой энергии и % времени
- **Сессия 3 — План 70/20/10:** персональный план развития на основе двух предыдущих сессий

По каждой сессии — отчёт от GPT-4o, скачивание в `.md`, оценка со звёздами.

## Стек

| Компонент | Решение |
|---|---|
| Бэкенд | Python + FastAPI |
| Фронтенд | HTML + CSS (без фреймворков), Jinja2 |
| AI | OpenAI API (GPT-4o) |
| Хранение данных | localStorage (без БД) |
| Фидбек | Telegram Bot |
| Деплой | Render |

## Запуск локально

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Создай `.env` в корне:
```
OPENAI_API_KEY=...
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

## Деплой

Render подхватывает изменения автоматически при пуше в `main`. Конфиг: `render.yaml`.

Переменные окружения добавить вручную в Render Dashboard:
- `OPENAI_API_KEY`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
