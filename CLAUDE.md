# AI Career Coach — контекст проекта

## Что это

Веб-приложение: пользователь проходит структурированную карьерную диагностику (форма), AI синтезирует ответы и возвращает персональный отчёт. Не чат, не диалог — одна сессия, один результат.

Цель: портфолио-проект, демонстрирующий продуктовое мышление (проблема → MVP → обратная связь → итерации).

## Аудитория

IT-специалисты (джуны и мидлы), которые застряли на текущей позиции.

## Стек

| Компонент | Решение |
|---|---|
| Бэкенд | Python + FastAPI |
| Фронтенд | HTML + CSS (без фреймворков), Jinja2-шаблоны |
| AI | OpenAI API (GPT-4o) |
| БД | Нет — данные живут только в запросе |
| Деплой | Render (render.yaml в корне) |
| Репо | https://github.com/Demonk03/AI-Career-sprint |

## Структура

```
main.py           — FastAPI-приложение, роуты
prompts.py        — построение промпта для OpenAI
templates/        — HTML-шаблоны (Jinja2)
  index_minimal.html  — лендинг
  survey.html         — многошаговая форма (4 блока)
  result.html         — страница результата
static/           — CSS
render.yaml       — конфиг деплоя на Render
.env              — OPENAI_API_KEY (не в git)
```

## UX-флоу

Лендинг → Форма (4 блока, прогресс-бар) → POST /analyze → OpenAI → Страница результата

### Блоки формы
1. Цели — куда хочешь прийти через 6–9 месяцев и почему
2. Ресурсы — компетенции, опыт, репутация
3. Личная эффективность — когда продуктивен, что не работает, что выбивает
4. SWOT — сильные/слабые стороны, возможности, угрозы

### Структура отчёта
- Точка 0 (текущее состояние)
- Сильные стороны
- Пробелы
- Рекомендации (3–5 шагов)
- Кнопка скачать результат
