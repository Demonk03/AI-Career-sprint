# AI Career Coach MVP — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Веб-приложение с 4-шаговой формой карьерной диагностики и AI-синтезом результата на странице + скачивание.

**Architecture:** FastAPI-сервер отдаёт статический HTML и принимает POST с ответами формы. Все 4 шага формы управляются JS на клиенте (прогресс-бар, переключение шагов). При сабмите — POST на `/analyze`, бэкенд передаёт данные в OpenAI и возвращает JSON с отчётом. Результат рендерится на той же странице.

**Tech Stack:** Python 3.11+, FastAPI, Jinja2, OpenAI Python SDK (gpt-4o), HTML/CSS vanilla JS, python-dotenv, markdown (для скачивания результата)

- [ ] **Step 2:** Обновить `/analyze` в `main.py`:
```python
from openai import OpenAI
from prompts import build_prompt

client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

@app.post("/analyze")
async def analyze(data: SurveyData):
    prompt = build_prompt(data.model_dump())
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": prompt}],
        temperature=0.7,
    )
    report_md = response.choices[0].message.content
    return {"report": report_md}
```

    <script>
        const report = localStorage.getItem('careerReport') || 'Отчёт не найден. Вернитесь и пройдите диагностику.';

        // Простой markdown → HTML (только заголовки и параграфы)
        function simpleMarkdown(text) {
            return text
                .replace(/^## (.+)$/gm, '<h2>$1</h2>')
                .replace(/^### (.+)$/gm, '<h3>$1</h3>')
                .replace(/^\* (.+)$/gm, '<li>$1</li>')
                .replace(/^- (.+)$/gm, '<li>$1</li>')
                .replace(/(<li>.*<\/li>\n?)+/g, '<ul>$&</ul>')
                .replace(/\n\n/g, '</p><p>')
                .replace(/^(?!<[h|u|l])(.+)$/gm, '<p>$1</p>');
        }

        document.getElementById('reportContent').innerHTML = simpleMarkdown(report);

        function downloadReport() {
            const blob = new Blob([report], { type: 'text/markdown' });
            const url = URL.createObjectURL(blob);
            const a = document.createElement('a');
            a.href = url;
            a.download = 'career-report.md';
            a.click();
        }
    </script>
</body>
</html>
